## Deployment

This website is optimized for deployment on Vercel.

### Deploy to Vercel

1. Push your code to GitHub
2. Import your repository to Vercel
3. Vercel will automatically detect and deploy your site

Or use the Vercel CLI:

```bash
npm i -g vercel
vercel
```
